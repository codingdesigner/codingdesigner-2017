    long-shadow

    @function long-shadow($direction, $length, $color, $fade: false, $shadow-count: 100) { ... }

    Description

    Function to generate long shadows (because flat is so has-been). Property-agnostic: works for both box-shadow and text-shadow. cos and sin might need to be polyfilled if Compass or any equivalent such as SassyMath is not in use.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $direction

    Shadow's direction (angle or keyword)

    		Direction— none
    $length

    Shadow's length

    		Length— none
    $color

    Shadow's color

    		Color— none
    $fade

    Whether or not shadow should fade:

    - `false` means no fading, shadow is `$color`
- `true`  means fading from `$color` to transparent
- a color means fading from `$color` to `$fade`
    		Bool or Colorfalse
    $shadow-count

    Number of computed shadows

    		Number100

    Returns

    List —

    List of shadows

    Example

    Usage

    .foo {
  text-shadow: long-shadow(42deg, 1em, #16a085);
}
.bar {
  box-shadow: long-shadow(to top left, 150px, hotpink, tomato);
}

    Requires

    • [function] sin
    • [function] cos
    • [external] Compass/helpers/math/cos — http://compass-style.org/reference/compass/helpers/math/#cos
    • [external] Compass/helpers/math/sin — http://compass-style.org/reference/compass/helpers/math/#sin

    Links

    • Pure Sass `cos` and `sin`

    Author

    • Hugo Giraudel
